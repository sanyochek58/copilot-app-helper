# Copilot App Helper

Фуллстек-приложение, которое даёт владельцу микробизнеса своего **ИИ-копилота**:  
помогает отвечать клиентам, писать письма сотрудникам, придумывать идеи и формулировать тексты.

Проект состоит из двух backend-сервисов на Spring Boot, frontend-приложения на React/Vite и Nginx в качестве reverse-proxy. Всё это можно поднять одной командой через Docker Compose.

## Архитектура

Проект состоит из нескольких сервисов:

- **`auth-service`** — сервис авторизации и аутентификации пользователей.
  - Отвечает за регистрацию/логин, выдачу токенов и проверку прав.
- **`ai-service`** — сервис ИИ-копилота.
  - За счёт Spring AI ходит к LLM и даёт ответы по системному промпту  
    («Ты — ИИ-копилот для владельца микробизнеса…»).
  - Для проверки пользователя обращается к `auth-service`.
- **`CHAT-WEBAPP`** — фронтенд на React + Vite.
  - Интерфейс для общения с копилотом.
- **`nginx`** (через `nginx.conf`) — reverse proxy.
  - Отдаёт статику фронта.
  - Проксирует запросы `/api/...` на backend-сервисы.

Все сервисы склеиваются сетью внутри `docker-compose.yml` и поднимаются одной командой.

## Технологии

**Backend**

- Java / Spring Boot
- Spring Web, Spring Security (в `auth-service`)
- Spring AI (в `ai-service`) для интеграции с LLM
- WebClient для общения между сервисами

**Frontend**

- React
- TypeScript
- Vite

**Инфраструктура**

- Docker & Docker Compose
- Nginx (reverse proxy)
